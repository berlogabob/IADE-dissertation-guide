# 05 — Citations and References

## Overview

Proper citation is essential for academic integrity. This guide covers APA style, the most common format for dissertations at IADE.

## Why Cite?

- Give credit to original authors
- Allow readers to find your sources
- Demonstrate the breadth of your research
- Avoid plagiarism

## In-Text Citations (APA 7th Edition)

### Basic Format
- **One author**: (Smith, 2020)
- **Two authors**: (Smith & Jones, 2020)
- **Three or more authors**: (Smith et al., 2020)
- **No author**: (Title of Work, 2020)
- **Multiple works**: (Smith, 2020; Jones, 2019)

### Narrative Citations
- Smith (2020) found that...
- According to Smith and Jones (2020)...

### Direct Quotes
- Include page number: (Smith, 2020, p. 45)

## Reference List

### Journal Article
```
Author, A. A., & Author, B. B. (Year). Title of article. Title of Periodical, volume(issue), pages. https://doi.org/xxxxx
```

### Book
```
Author, A. A. (Year). Title of work: Capital letter also for subtitle. Publisher.
```

### Website
```
Author, A. A. (Year, Month Day). Title of page. Site Name. URL
```

### Conference Paper
```
Author, A. A. (Year, Month Day–Day). Title of paper. Conference Name, Location. Publisher.
```

## Using Zotero

### Setup
1. Install [Zotero](https://www.zotero.org/) (desktop app)
2. Install the browser connector
3. Set output style to **APA 7th Edition**

### Workflow
1. Save papers to Zotero using the browser connector
2. Organize with collections and tags
3. Use the Word/LibreOffice plugin to insert citations
4. Generate bibliography automatically

### Tips
- Use **ZoteroBib** (zbib.org) for quick one-off bibliographies
- Sync your library across devices with a free Zotero account
- Add PDFs to Zotero for full-text search

See: [Bibliography Managers](../resources/bibliography-managers.md)

## Common Mistakes

| Mistake | Solution |
|---------|----------|
| Inconsistent citation style | Set Zotero to APA and stick to it |
| Missing DOIs | Always include DOI when available |
| Citing in text but not in references | Every in-text citation needs a reference |
| Reference in list but not cited | Remove unused references |
| Wrong order of authors | Follow APA rules exactly |
