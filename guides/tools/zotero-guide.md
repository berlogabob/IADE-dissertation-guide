# Zotero Guide

## What is Zotero?

Zotero is a free, open-source tool for collecting, organizing, and citing research sources. It works as a desktop app with a web version and browser connector.

## Installation

1. Download from [zotero.org](https://www.zotero.org/)
2. Install the **Zotero Connector** for your browser (Chrome, Firefox, Edge)
3. Create a free account for syncing

## Basic Workflow

### Adding Sources

- **Browser Connector**: Click the Zotero icon while viewing a paper → saves it automatically
- **Manual**: Click the green "+" button → add by DOI, ISBN, or URL
- **PDF drag-and-drop**: Drag a PDF into Zotero → it extracts metadata automatically

### Organizing

- **Collections**: Create folders for different topics or chapters
- **Tags**: Add keywords for easy searching
- **Related**: Link papers that reference each other

### Citing in Word/LibreOffice

1. Install the Zotero word plugin (Tools → Add-ons)
2. Place cursor where you want the citation
3. Click "Add/Edit Citation" in the Zotero toolbar
4. Search for the reference → press Enter
5. Zotero inserts the formatted citation

### Generating Bibliography

1. Click "Add/Edit Bibliography" in the toolbar
2. Zotero creates a formatted reference list

## Setting APA Style

1. Edit → Preferences → Cite
2. Click "Get Additional Styles"
3. Search "APA 7th Edition"
4. Install and select it

## Tips

- **Use groups** to share libraries with collaborators
- **Sync PDFs** across devices with a free account
- **Use ZoteroBib** (zbib.org) for quick one-off bibliographies without the full app
