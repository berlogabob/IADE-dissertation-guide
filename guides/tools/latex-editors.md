# LaTeX Editors

Tools for writing and compiling LaTeX documents.

## Online Editors

### [Overleaf](https://www.overleaf.com/)
- **Type**: Web-based
- **Cost**: Free (with limits) / Premium
- **Best for**: Beginners, collaboration
- **Features**: Real-time preview, collaboration, Git integration, templates
- **Platforms**: Any browser

### [ChatGPT Prism](https://prism.openai.com/)
- **Type**: Web-based with AI
- **Cost**: Requires OpenAI account
- **Best for**: AI-assisted LaTeX writing
- **Features**: AI writing assistance, code completion, debugging help
- **Platforms**: Any browser

## Desktop Editors

### [TeXstudio](https://www.texstudio.org/)
- **Type**: Desktop app
- **Cost**: Free, open source
- **Best for**: Most users
- **Features**: Syntax highlighting, auto-completion, built-in viewer, spell check
- **Platforms**: Windows, macOS, Linux

### [VS Code + LaTeX Workshop](https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop)
- **Type**: VS Code extension
- **Cost**: Free
- **Best for**: Developers familiar with VS Code
- **Features**: IntelliSense, compilation, PDF viewer, snippets
- **Platforms**: Windows, macOS, Linux

## LaTeX Distributions

These provide the LaTeX compiler (pdflatex, biber, etc.).

### [MiKTeX](https://miktex.org/download)
- **Type**: LaTeX distribution
- **Cost**: Free
- **Best for**: Windows users
- **Features**: Automatic package installation, easy setup
- **Platforms**: Windows

### [TeX Live](https://www.tug.org/texlive/)
- **Type**: LaTeX distribution
- **Cost**: Free
- **Best for**: Linux/macOS users, full installation
- **Features**: Complete collection of packages, cross-platform
- **Platforms**: Windows, macOS, Linux

## Recommended Setup

### For beginners
1. Use **Overleaf** — no installation needed
2. Or install **MiKTeX** + **TeXstudio** on Windows

### For developers
1. Install **TeX Live** (or MiKTeX on Windows)
2. Use **VS Code** with LaTeX Workshop extension

### For AI-assisted writing
1. Use **ChatGPT Prism** for AI help
2. Compile locally or on Overleaf
