# Bibliotecas Digitais e Repositórios

Bases de dados académicas para encontrar artigos revistos por pares, atas de conferências e relatórios técnicos.

## Pesquisa Geral

| Nome | Notas |
|------|-------|
|  <a href="https://scholar.google.com/" target="_blank" rel="noopener noreferrer">Google Scholar</a>| Cobertura ampla, gratuito |
|  <a href="https://scholar.google.com/scholar_labs/search" target="_blank" rel="noopener noreferrer">Google Scholar AI</a> | Scholar Labs | Pesquisa melhorada com IA |

## Dicas

- **Pesquise em múltiplas bases de dados** → Nenhuma fonte única cobre tudo
- **Use pesquisa avançada** → Combine palavras-chave com AND, OR, NOT
- **Configure alertas** → Seja notificado quando novos artigos correspondem à sua query
- **Verifique citações** → Siga a cadeia de citações para a frente e para trás
- **Use empréstimo entre bibliotecas** → Solicite artigos que a sua biblioteca não possui
