# Guia do Zotero

## O que é o Zotero?

O Zotero é uma ferramenta gratuita e de código aberto para recolher, organizar e citar fontes de investigação. Funciona como aplicação desktop com versão web e conetor de navegador.

## Instalação

1. Transferir de [zotero.org](https://www.zotero.org/)
2. Instalar o **Conetor Zotero** para o seu navegador (Chrome, Firefox, Edge)
3. Criar uma conta gratuita para sincronização

## Fluxo de Trabalho Básico

### Adicionar Fontes

- **Conetor do Navegador**: Clique no ícone do Zotero ao visualizar um artigo → guarda automaticamente
- **Manual**: Clique no botão verde "+" → adicionar por DOI, ISBN ou URL
- **Arrastar PDF**: Arraste um PDF para o Zotero → extrai metadados automaticamente

### Organizar

- **Coleções**: Crie pastas para diferentes temas ou capítulos
- **Etiquetas**: Adicione palavras-chave para facilitar a pesquisa
- **Relacionados**: Ligue artigos que se referem uns aos outros

### Citar no Word/LibreOffice

1. Instalar o plugin do Zotero (Ferramentas → Extras)
2. Colocar o cursor onde pretende a citação
3. Clicar em "Adicionar/Editar Citação" na barra de ferramentas do Zotero
4. Pesquisar a referência → premir Enter
5. O Zotero insere a citação formatada

### Gerar Bibliografia

1. Clicar em "Adicionar/Editar Bibliografia" na barra de ferramentas
2. O Zotero cria uma lista de referências formatada

## Definir Estilo APA

1. Editar → Preferências → Citar
2. Clicar em "Obter Estilos Adicionais"
3. Pesquisar "APA 7th Edition"
4. Instalar e selecionar

## Dicas

- **Use grupos** para partilhar bibliotecas com colaboradores
- **Sincronize PDFs** entre dispositivos com uma conta gratuita
- **Use o ZoteroBib** (zbib.org) para bibliografias rápidas pontuais sem a aplicação completa
