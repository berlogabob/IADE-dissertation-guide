# Editores LaTeX

Ferramentas para escrever e compilar documentos LaTeX.

## Editores Online

### [Overleaf](https://www.overleaf.com/)
- **Tipo**: Baseado na web
- **Custo**: Gratuito (com limites) / Premium
- **Mais adequado para**: Principiantes, colaboração
- **Funcionalidades**: Pré-visualização em tempo real, colaboração, integração Git, modelos
- **Plataformas**: Qualquer navegador

### [ChatGPT Prism](https://prism.openai.com/)
- **Tipo**: Baseado na web com IA
- **Custo**: Gratuito, mas requer conta OpenAI
- **Mais adequado para**: Principiantes, colaboração e escrita LaTeX assistida por IA
- **Funcionalidades**: Assistência de escrita por IA, completar código, ajuda na depuração
- **Plataformas**: Qualquer navegador

## Editores Desktop

### [TeXstudio](https://www.texstudio.org/)
- **Tipo**: Aplicação desktop
- **Custo**: Gratuito, código aberto
- **Mais adequado para**: Maioria dos utilizadores
- **Funcionalidades**: Destaque de sintaxe, auto-completar, visualizador integrado, verificação ortográfica
- **Plataformas**: Windows, macOS, Linux

### [VS Code + LaTeX Workshop](https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop)
- **Tipo**: Extensão do VS Code
- **Custo**: Gratuito
- **Mais adequado para**: Programadores familiarizados com VS Code
- **Funcionalidades**: IntelliSense, compilação, visualizador PDF, snippets
- **Plataformas**: Windows, macOS, Linux

## Distribuições LaTeX

Estas fornecem o compilador LaTeX (pdflatex, biber, etc.).

### [MiKTeX](https://miktex.org/download)
- **Tipo**: Distribuição LaTeX
- **Custo**: Gratuito
- **Mais adequado para**: Utilizadores Windows
- **Funcionalidades**: Instalação automática de pacotes, configuração fácil
- **Plataformas**: Windows

### [TeX Live](https://www.tug.org/texlive/)
- **Tipo**: Distribuição LaTeX
- **Custo**: Gratuito
- **Mais adequado para**: Utilizadores Linux/macOS, instalação completa
- **Funcionalidades**: Coleção completa de pacotes, multiplataforma
- **Plataformas**: Windows, macOS, Linux

## Configuração Recomendada

### Para principiantes
1. Use **Overleaf** ou **Prism** - sem necessidade de instalação
2. Ou instale **MiKTeX** + **TeXstudio** no Windows

### Para programadores
1. Instale **TeX Live** (ou **MiKTeX** no Windows)
2. Use **VS Code** com a extensão LaTeX Workshop

### Para escrita assistida por IA
1. Use **ChatGPT Prism** para ajuda de IA
