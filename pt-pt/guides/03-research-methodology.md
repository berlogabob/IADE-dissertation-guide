# 03 - Metodologia de Investigação

## Visão Geral

O capítulo de metodologia explica **como** realizou a sua investigação. Deve ser rigoroso o suficiente para que outro investigador possa replicar o seu estudo.

## Tipos de Investigação

### Quantitativa

- Utiliza **dados numéricos** e análise estatística
- Testa hipóteses através de experiências, inquéritos ou conjuntos de dados existentes
- Métodos comuns: experiências, inquéritos, análise estatística

### Qualitativa

- Utiliza **dados não numéricos** (entrevistas, observações, documentos)
- Explora significados, experiências e perspetivas
- Métodos comuns: estudos de caso, entrevistas, etnografia, teoria fundamentada

### Métodos Mistos

- Combina abordagens quantitativas e qualitativas
- Proporciona uma compreensão mais abrangente
- Requer justificação para a combinação de métodos

## Passos para Definir a Metodologia

### 1. Escolha a Sua Abordagem

Com base na sua questão de investigação:
- **Exploratória** → Qualitativa
- **Descritiva** → Ambas (depende dos dados)
- **Explicativa** → Quantitativa
- **Abrangente** → Métodos Mistos

### 2. Defina a Recolha de Dados

| Método | Quando Utilizar |
|--------|-----------------|
| Inquérito/Questionário | Amostra grande, dados padronizados |
| Entrevistas | Compreensão aprofundada, amostra pequena |
| Experiências | Testar relações causa-efeito |
| Estudo de Caso | Análise aprofundada de caso específico |
| Dados Secundários | Utilizar conjuntos de dados existentes |

### 3. Defina a Sua Amostra

- **População** → Quem/o que está a estudar
- **Método de amostragem** → Como seleciona os participantes
- **Tamanho da amostra** → Quantos (justifique a escolha)

### 4. Defina a Análise

- **Quantitativa**: Testes estatísticos (t-test, ANOVA, regressão, etc.)
- **Qualitativa**: Análise temática, codificação, análise de conteúdo

### 5. Aborde a Validade e Fiabilidade

- **Validade interna** → Resultados devem-se à sua intervenção, não a fatores confundentes
- **Validade externa** → Resultados generalizáveis para outros contextos
- **Fiabilidade** → Resultados consistentes e reproduzíveis

### 6. Considerações Éticas

- Consentimento informado
- Proteção de dados e privacidade
- Anonimato e confidencialidade
- Aprovação institucional (se necessário)

## Dicas de Escrita

- Escreva no **passado** (já realizou)
- Seja específico em cada detalhe (ferramentas, configurações, datas)
- Justifique cada escolha metodológica
- Reconheça limitações honestamente

## Erros Comuns

| Erro | Solução |
|------|---------|
| Métodos não correspondem à questão | Reajustar a abordagem à questão |
| Sem justificação para tamanho da amostra | Usar análise de potência ou citar estudos semelhantes |
| Descrição vaga da análise | Ser específico em cada passo |
| Ignorar limitações | Abordá-las diretamente neste capítulo |
