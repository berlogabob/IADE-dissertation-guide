# 01 - Escolher o Tema

## Visão Geral

Escolher o tema da dissertação é uma das decisões mais importantes no seu percurso de mestrado. Um tema bem definido estabelece a base para toda a sua investigação.

## Passos

### 1. Identifique os Seus Interesses

- Quais temas genuinamente despertam o seu interesse?
- Quais disciplinas mais gostou?
- Quais problemas do seu campo deseja resolver?

### 2. Reveja a Literatura Atual

Consulte publicações recentes para identificar:
- **Lacunas de investigação** → O que ainda não foi explorado?
- **Tendências emergentes** → Quais temas estão a ganhar relevância?
- **Debates não resolvidos** → Onde os especialistas discordam?

Utilize [bibliotecas digitais](../resources/digital-libraries.md) e [ferramentas de IA](../resources/ai-tools.md) para explorar.

### 3. Avalie a Viabilidade

Questione-se:
- Posso completar isto dentro do prazo?
- Tenho acesso aos dados/recursos necessários?
- O âmbito é apropriado para uma dissertação de mestrado?

### 4. Formule uma Questão de Investigação

Uma boa questão de investigação é:
- **Clara** → Específica e inequívoca
- **Focada** → Suficientemente delimitada para ser gerível
- **Complexa** → Requer análise, não apenas descrição
- **Relevante** → Significativa para o seu campo

### 5. Escreva um Título Preliminar

O seu título deve:
- Refletir o âmbito do seu estudo
- Incluir variáveis ou conceitos-chave
- Ser conciso (máximo 10–15 palavras)

## Erros Comuns

| Erro | Solução |
|------|---------|
| Tema demasiado amplo | Delimitar para um aspeto específico |
| Tema demasiado restrito | Garantir que existe literatura suficiente |
| Sem questão de investigação clara | Refinar até poder enunciar numa frase |
| Escolha baseada apenas em tendências | Equilibrar interesse com viabilidade |
